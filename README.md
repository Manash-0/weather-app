## 📦 Installation

```bash
git clone https://github.com/Manash-0/weather-app.git
cd weather-app
npm install
```
🔐 Environment Variables
Create a .env file in the root with your API key:
```bash
API_KEY=your_openweathermap_api_key
```
RUN:
```bash
node server.js
```
